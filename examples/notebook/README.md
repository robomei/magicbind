# notebook example

Write C++ inline in Jupyter. Magicbind compiles and injects the module immediately.

**Local** (magicbind already installed in your environment):
```bash
uv run jupyter notebook magicbind_demo.ipynb
```

**Colab**: open `magicbind_colab.ipynb` — includes `pip install` and `apt-get` cells.
